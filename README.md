# Building 

## Requirments 
Python >= 3.9 

## Create a venv
python -m venv venv 
source venv/bin/activate

## Install Deps
pip install -r requirements.txt 

## Create Windows Installer
(use git bash)
bash build_win.sh

## Create Linux Installer
bash build_linux.sh

## Create MacOS Insatller
bash build_macos.sh
